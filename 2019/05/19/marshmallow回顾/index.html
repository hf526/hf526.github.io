marshmallow 回顾

marshmallow 是一个序列化的库，可以实现object -> dict ， objects -> list， string -> dict和 string -> list ，之前开发的demo，进行序列化比较麻烦，后面用过marshmallow 一下，由于太久没有了，今天就看了一下用法

#####用法
首先是原始类，然后定义一个序列化类，然后生成了一个原始类对象，实例化序列化类，用dumps/dump即可完成对象转字典或者str的序列化，反序列化使用load即可

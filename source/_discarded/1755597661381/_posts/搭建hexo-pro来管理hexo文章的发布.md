#### **Hexo Pro**（现代化推荐）

* **功能**：文章/页面管理、图床集成、可视化配置编辑、全局搜索、多用户权限**1**2**4**。
* **安装**：

  先卸载 npm remove hexo-admin
  再安装
  npm install --save hexo-pro
  hexo server -d

  **访问**：`http://localhost:4000/pro/`

  **优势**：React 前端，支持多主题切换和移动端适配
